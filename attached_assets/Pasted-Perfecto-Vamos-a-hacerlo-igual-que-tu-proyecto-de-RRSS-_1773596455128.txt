PROMPT MAESTRO – Sistema Web Farmacia con Google Sheets API
Quiero crear una plataforma web para gestión de farmacia utilizando HTML, CSS y JavaScript puro, conectada directamente a Google Sheets mediante Google Sheets API, sin Apps Script.
________________________________________
Objetivo del sistema
Crear un POS Web para farmacia que permita:
•	buscar productos
•	vender productos
•	manejar fiados
•	registrar movimientos
•	ver dashboard
•	administrar usuarios
Todo usando Google Sheets como base de datos.
